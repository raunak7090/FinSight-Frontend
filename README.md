# Insightful Wallets Frontend

# Overview

This repository contains the Insightful Wallets frontend built with Vite, React, TypeScript, and Tailwind CSS.

## Getting Started

1. Ensure Node.js and your preferred package manager (npm, pnpm, or bun) are installed.
2. Clone the repository and install dependencies:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
